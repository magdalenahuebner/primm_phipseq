
# PRIMM PhIP-Seq Analysis Pipeline  
### Predicting Response to Immunotherapy for Melanoma With Gut Microbiome

<!-- badges: start -->
<!-- badges: end -->

This repository contains a small, frozen analysis pipeline for PhIP-Seq data
generated from melanoma patient samples.

The aim of the project is to study antibody response patterns in melanoma and to
investigate their association with clinical response to immune checkpoint
inhibitor therapy.
